# Joint Editing Development Documentation

## Instructions:

1. ``npm install``

1. ``npm run bootstrap``

1. ``npm run build``

1. ``npm run start``

1. If your browser doesn't open application, please do it manually with the link [http://localhost:4200](http://localhost:4200).
